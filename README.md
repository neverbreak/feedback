# neverbreak.ai feedback

Public issue tracker for [neverbreak.ai](https://neverbreak.ai) — CI broke, AI fixed, with proof.

Open an issue if you've:

- found a **bug** — something broke or behaved unexpectedly
- got a **feature request** — what would make this more useful for you
- have a **question** — about how it works, what it can do, or what it doesn't

Every issue gets read. Replies usually within a day.

For private/security issues: email `support@neverbreak.ai` directly.
